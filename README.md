# 👔 متجر الملابس الإلكترونية - Clothing Store

متجر ملابس احترافي وحديث مع نظام شات مباشر مع البائع

## ✨ المميزات الرئيسية

✅ **عرض المنتجات بالصور والسعر** - تصميم جميل وسهل الاستخدام
✅ **بدون تسجيل دخول** - يمكن للعملاء التسوق مباشرة
✅ **نظام الشات الفوري** - التواصل المباشر مع البائع في الوقت الفعلي
✅ **سلة التسوق** - حفظ المنتجات والشراء بسهولة
✅ **نظام الطلبات** - معالجة وتتبع الطلبات
✅ **لوحة التحكم** - إدارة الرسائل والطلبات
✅ **تصميم متجاوب** - يعمل على جميع الأجهزة

## 📋 المتطلبات

- Python 3.8+
- Flask
- Flask-CORS

## 🚀 التثبيت والتشغيل

### 1. تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 2. تشغيل التطبيق
```bash
python app.py
```

### 3. فتح المتجر
```
http://localhost:5000
```

### 4. لوحة التحكم (البائع)
```
http://localhost:5000/admin
```

## 🛍️ كيفية الاستخدام

### للعميل:
1. تصفح المنتجات من الصفحة الرئيسية
2. ابحث عن منتجات معينة
3. اضغط على "التفاصيل" لرؤية تفاصيل المنتج
4. اضغط على "أضف للسلة" لإضافة المنتج
5. اضغط على "السلة" لمراجعة الطلب
6. أدخل بيانات التسليم واضغط "إتمام الشراء"
7. استخدم الشات 💬 للتواصل مع البائع

### للبائع (لوحة التحكم):
1. ادخل إلى `/admin`
2. شاهد جميع الرسائل من العملاء
3. شاهد جميع الطلبات
4. أرسل رسائل للعملاء
5. حدث حالة الطلب (قيد الانتظار، مؤكد، قيد الشحن، تم التسليم)

## 💾 قاعدة البيانات

يتم حفظ البيانات تلقائياً في قاعدة بيانات SQLite

### الجداول:
- **products** - المنتجات
- **messages** - الرسائل والشات
- **orders** - الطلبات
- **order_items** - تفاصيل الطلبات
